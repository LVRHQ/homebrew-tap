# LVRHQ Homebrew tap

Internal Homebrew formulae for LocalVR tooling.

## Install

```bash
brew install LVRHQ/tap/wdydt-agent
```

## Formulae

- **wdydt-agent** — Mac agent for [lvr-wdydt-app](https://github.com/LVRHQ/lvr-wdydt-app). Pushes ActivityWatch buckets and Claude Code session logs to https://wdydt.golocalvr.com.
